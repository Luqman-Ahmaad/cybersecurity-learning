# 🌐 OSI Model (Open Systems Interconnection)

## 🎯 Objective
To understand the 7 layers of the OSI model and their role in network communication.

---

## 🧠 What is the OSI Model?

The OSI Model is a conceptual framework used to understand how data travels across a network.

It divides communication into **7 layers**, each with a specific function.

---

## 🧱 OSI Layers (Top to Bottom)

| Layer | Name | Function |
|------|------|----------|
| 7 | Application | User interface & applications |
| 6 | Presentation | Data formatting & encryption |
| 5 | Session | Session management |
| 4 | Transport | Data delivery & reliability |
| 3 | Network | Routing & IP addressing |
| 2 | Data Link | MAC addressing & switching |
| 1 | Physical | Transmission of raw bits |

---

## 🔍 Detailed Explanation

### 🟣 Layer 7: Application
- Closest to user
- Examples: HTTP, FTP, SMTP

---

### 🔵 Layer 6: Presentation
- Data encryption/decryption
- Data compression

---

### 🟢 Layer 5: Session
- Manages sessions between systems
- Opens/closes communication

---

### 🟡 Layer 4: Transport
- Ensures reliable delivery
- Protocols: TCP, UDP

---

### 🟠 Layer 3: Network
- Handles routing
- Uses IP addresses

---

### 🔴 Layer 2: Data Link
- Uses MAC addresses
- Error detection

---

### ⚫ Layer 1: Physical
- Cables, signals, hardware

---


---

## 🔐 Importance in Cybersecurity

- Helps identify where attacks occur
- Example:
  - DDoS → Network layer
  - XSS → Application layer

---

## 🧠 What I Learned

- Network communication is layered
- Each layer has a specific role
- Attacks target different layers

---

## ✅ Conclusion

The OSI model is essential for understanding networking and analyzing cyber attacks effectively.
