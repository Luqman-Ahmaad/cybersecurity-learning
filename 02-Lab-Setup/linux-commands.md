# 🐧 Essential Linux Commands for Cybersecurity

## 🎯 Objective
To learn fundamental Linux commands used in cybersecurity and penetration testing.

---

## 🧠 Why Linux?

Most hacking tools run on Linux.  
Understanding Linux is critical for:
- Navigating systems
- Running tools
- Exploiting vulnerabilities

---

## 📁 File & Directory Commands

| Command | Description |
|--------|------------|
| pwd | Show current directory |
| ls | List files |
| cd | Change directory |
| mkdir | Create folder |
| rm | Delete files |
| cp | Copy files |
| mv | Move/rename files |

---

## 📄 Viewing Files

| Command | Description |
|--------|------------|
| cat | View file content |
| less | Scroll through file |
| head | First lines |
| tail | Last lines |

---

## 🔍 Searching

| Command | Description |
|--------|------------|
| find | Search files |
| grep | Search text inside files |

Example:

grep "password" file.txt


---

## 🔐 Permissions

| Command | Description |
|--------|------------|
| chmod | Change permissions |
| chown | Change ownership |

Example:

chmod 777 file.txt


---

## ⚙️ System Info

| Command | Description |
|--------|------------|
| whoami | Current user |
| uname -a | System info |
| top | Running processes |
| ps aux | Process list |

---

## 🌐 Networking Commands

| Command | Description |
|--------|------------|
| ifconfig / ip a | Show IP |
| ping | Test connection |
| netstat | Network connections |
| ss | Socket statistics |
