# 🖥️ Virtual Machine Setup (Lab Environment)

## 🎯 Objective
To set up a safe and isolated lab environment for practicing cybersecurity techniques.

---

## 🧠 What is a Virtual Machine?

A Virtual Machine (VM) is a software-based computer that runs inside your physical system.

It allows you to:
- Run multiple operating systems
- Test tools safely
- Perform hacking labs without risk

---

## 🛠️ Tools Used

- VirtualBox / VMware
- Kali Linux ISO

---

## ⚙️ Why Use a VM for Cybersecurity?

- Isolation from host system
- Safe environment for testing malware/tools
- Easy to reset (snapshots)
- No risk to main OS

---

## 🔧 Steps to Setup VM

### 1. Install Virtualization Software
- Download and install:
  - VirtualBox OR VMware

---

### 2. Download Kali Linux

- Go to official site:
  https://www.kali.org/get-kali/

- Download:
  - Kali Linux ISO

---

### 3. Create New Virtual Machine

- Open VirtualBox / VMware
- Click **New VM**
- Select:
  - OS: Linux
  - Version: Debian (64-bit)

---

### 4. Allocate Resources

- RAM: 4GB (minimum)
- CPU: 2 cores
- Storage: 40GB+

---

### 5. Attach ISO File

- Select downloaded Kali Linux ISO
- Boot from ISO

---

### 6. Install Kali Linux

- Start VM
- Follow installation steps
- Set:
  - Username
  - Password

---

### 7. Install Guest Additions (Important)

- Improves performance
- Enables:
  - Full screen
  - Clipboard sharing

---

## 🔐 Safety Notes

- Never attack real systems
- Only use lab environments
- Keep VM isolated
