# 🌐 Network Basics

## 🎯 Objective
To understand the fundamental concepts of computer networking used in cybersecurity.

---

## 🧠 What is a Network?

A network is a group of connected devices that communicate with each other.

Examples:
- Internet
- Local Area Network (LAN)
- Wi-Fi networks

---

## 🖥️ Types of Networks

| Type | Description |
|------|------------|
| LAN | Local Area Network (small area) |
| WAN | Wide Area Network (large scale) |
| MAN | Metropolitan Area Network |
| PAN | Personal Area Network |

---

## 🔗 Network Components

- **Router** → Connects networks  
- **Switch** → Connects devices in LAN  
- **Firewall** → Filters traffic  
- **Server** → Provides services  
- **Client** → Requests services  

---

## 📡 How Data Travels

1. Data is broken into packets  
2. Packets travel through network  
3. Reassembled at destination  

---

## 🌐 IP Address

An IP address uniquely identifies a device on a network.

Example:

192.168.1.1


Types:
- Private IP
- Public IP

---

## 🔄 MAC Address

- Unique hardware address
- Assigned to network interface

Example:

00:1A:2B:3C:4D:5E


---

## 📦 Protocols

Rules for communication:
- HTTP → Web
- FTP → File transfer
- DNS → Domain resolution

---

## 🔍 DNS (Domain Name System)

Converts:

google.com → IP address


---

## 🔐 Importance in Cybersecurity

- Helps identify network vulnerabilities  
- Required for scanning and attacks  
- Used in traffic analysis  

---

## 🧠 What I Learned

- Networks connect devices and enable communication  
- IP and MAC addresses identify devices  
- Protocols define how data is exchanged  

---

## ✅ Conclusion

Understanding network basics is essential for analyzing traffic, scanning systems, and performing cybersecurity operations.
